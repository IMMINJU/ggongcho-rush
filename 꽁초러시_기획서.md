# 꽁초러시 (Cigarette Butt Rush)

> "삶의 목적은 죽음" - 현대 사회의 소외된 존재, 중독의 굴레, 그리고 역설적으로 '죽음'만이 유일한 해방인 비극을 그린 게임

---

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| **장르** | 서바이벌 / 스텔스 |
| **뷰** | 탑다운 2D |
| **플랫폼** | 웹 브라우저 (Canvas 기반) |
| **톤 & 무드** | 다크, 우울, 사회비판적 |

### 핵심 콘셉트
니코틴 중독자가 돈 없이 길거리에서 담배꽁초를 주워 피며 연명하는 서바이벌 게임.
시야가 100% 열리면 죽음으로 게임 클리어 - 역설적 해방.

---

## 2. 핵심 시스템

### 2.1 니코틴 게이지
- 시간이 지나면 자동으로 감소
- 0%가 되면 **게임오버 (금단현상)**
- 꽁초를 피우면 충전
- 꽁초 길이에 따라 충전량 상이

| 꽁초 등급 | 충전량 | 출현 확률 |
|-----------|--------|-----------|
| 짧은 꽁초 | +5% | 50% |
| 보통 꽁초 | +15% | 35% |
| 장초 | +30% | 15% |

### 2.2 시야 시스템
플레이어 주변만 원형으로 보이고 나머지는 **암흑**.

```
■■■■■■■■■
■■■░░░■■■
■■░░░░░■■
■■░░●░░■■  ● = 플레이어
■■░░░░░■■  ░ = 시야
■■■░░░■■■  ■ = 암흑
■■■■■■■■■
```

- **니코틴 0~30%**: 시야 반경 최소 (좁고 답답함)
- **니코틴 30~60%**: 시야 중간
- **니코틴 60~90%**: 시야 넓음
- **니코틴 100%**: 시야 완전 오픈 → **죽음 (게임 클리어)**

### 2.3 도둑질 시스템
- 편의점에서 담배 도둑질 시도 가능
- 성공 시: 새 담배 획득 (니코틴 +50%)
- 실패 시 (경찰 발각): **게임오버 (체포)**

---

## 3. 맵 구성

**단일 맵: 도심 번화가 (루프형 구조)**

### 구역별 특징

| 구역 | 특징 | 흡연자 밀도 | 위험도 |
|------|------|-------------|--------|
| **술집 골목** | 취객 多, 장초 확률 높음 | 높음 | 낮음 |
| **편의점 앞** | 담배 도둑질 가능, 경찰 순찰 | 중간 | 높음 |
| **공원 벤치** | 꽁초 드랍 많음, 질 낮음 | 낮음 | 낮음 |
| **버스 정류장** | 대기 중 흡연자, 전자담배 함정 | 중간 | 중간 |
| **오피스 뒷골목** | 점심시간 흡연자 폭발 | 시간대별 | 중간 |

---

## 4. NPC

### 4.1 흡연자 유형

| 유형 | 흡연 속도 | 꽁초 품질 | 특징 |
|------|-----------|-----------|------|
| **직장인** | 빠름 | 중간 | 시간에 쫓기듯 핌 |
| **취객** | 느림 | 높음 (장초 확률↑) | 비틀거리며 천천히 |
| **학생** | 매우 빠름 | 낮음 | 몰래 피고 급히 버림 |
| **전자담배러** | - | 없음 | **함정!** 따라가면 시간낭비 |

### 4.2 경찰
- 편의점 주변 정기 순찰
- 시야 내에 있으면 도둑질 불가
- 도둑질 현장 발각 시 → **체포 엔딩**

### 4.3 경쟁 노숙자
- 플레이어와 같은 꽁초를 노림
- 먼저 도착한 쪽이 획득
- 가끔 힌트 제공: *"저기 아저씨 담배 거의 다 탔어"*

### 4.4 노점상
- 돈이 있으면 담배 구매 가능
- 돈 = 바닥 동전 줍기 (희귀)

## 5. 이벤트

| 이벤트 | 발생 조건 | 효과 |
|--------|-----------|------|
| **비 오는 날** | 랜덤 | 바닥 꽁초 대부분 사용 불가 (젖음) |
| **단속의 날** | 랜덤 | 흡연자 감소, 경찰 증가 |
| **회식 시즌** | 랜덤 | 술집 골목 흡연자 대폭 증가 |
| **행운의 장초** | 희귀 | 레어 꽁초 발견, 니코틴 대량 충전 |

## 6. 게임 흐름

```
┌─────────────────────────────────────┐
│          게임 시작                   │
│      니코틴 50%, 시야 중간           │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│          메인 루프                   │
│  • 맵 배회                          │
│  • 꽁초 수집 / 흡연자 스토킹         │
│  • 담배연기 추적                     │
│  • (선택) 도둑질 시도                │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│         꽁초 흡연                    │
│  • 니코틴 충전                       │
│  • 시야 확장                         │
└──────────────┬──────────────────────┘
               ↓
       ┌───────┴───────┐
       ↓               ↓
┌──────────────┐ ┌──────────────┐
│ 니코틴 < 100%│ │ 니코틴 = 100%│
│   계속 진행  │ │  시야 100%   │
└──────┬───────┘ └──────┬───────┘
       ↓               ↓
    메인 루프      게임 클리어
                   (죽음=해방)
```

## 7. 엔딩

| 엔딩 | 조건 | 연출 | 의미 |
|------|------|------|------|
| **게임오버 A** | 니코틴 0% | 화면 흔들림, 쓰러짐 | 금단현상 |
| **게임오버 B** | 경찰 체포 | 사이렌, 화면 암전 | 사회 낙오 |
| **클리어** | 니코틴 100% | 시야 완전 오픈, 서서히 하얗게 | 죽음으로 해방 |

### 클리어 엔딩 연출 (안)
1. 시야가 서서히 완전히 밝아짐
2. 모든 것이 보이는 순간
3. 화면이 하얗게 번아웃
4. 검은 화면에 텍스트:

> *"드디어, 볼 수 있게 되었다."*
>
> *"그리고 더 이상, 볼 필요가 없어졌다."*
